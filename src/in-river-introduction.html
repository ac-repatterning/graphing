RIVER LEVELS

This is a temporary live system. It predicts river levels and provides supplementary material, predominantly for officials aiding Scottish Government Resilience activities. A program scans for National Weather Service warnings.

During a warning period: If one or more gauge stations, on record, lie within a warning area, temporary schedules are automatically created. Automatically

  • All products, except the forecasts graphs, are updated every 2 hours; this can be every 3 hours, hour, etc.
  • Trying forecasts graphs updates for different schedules; focusing on catchments that host warning area gauges.

Outwith a warning period: Automatic daily updates of all products, except the forecasts graphs.

Future items possibilities include forecasts & rate-of-change activated alerts. For more information, including code, setup and configurations, etc. visit GitHub
